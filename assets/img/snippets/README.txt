Home-page snippet images
=========================

The six story snippets that orbit the heart on the home page each look for an
image here:

  1.jpg  2.jpg  3.jpg  4.jpg  5.jpg  6.jpg

Drop in photos with those exact names (landscape works best, roughly 600x400)
and they appear automatically — no code changes needed.

Until a file exists, that snippet shows a soft gold-on-ink gradient instead, so
the page always looks finished. To change which file a snippet uses, edit the
matching background-image url in index.html (search for "/assets/img/snippets/").
