Home-page snippet images
=========================

The six story snippets that orbit the heart on the home page each load one
image from this folder. Upload your art with these EXACT names so each picture
lines up with its caption:

  1.jpg  ->  "Pit Stop"            (Eddie & Ginnie at the Circle K)
  2.jpg  ->  "Back from California" (Ghost & Eddie in the studio)
  3.jpg  ->  "Instant Connection"  (Dozier the puppy / living room)
  4.jpg  ->  "The Lake"            (beach campfire)
  5.jpg  ->  "Go-Karts, Redux"     (go-kart track)
  6.jpg  ->  "Theory Class, IRL"   (music theory lecture)

Landscape images look best (roughly 600x400). Until a file exists, that snippet
shows a soft gold-on-ink gradient, so the page always looks finished.

To change which file a snippet uses or its caption, edit index.html
(search for "/assets/img/snippets/").
